# Sesión 02 — Puerta en Valla y Figurante Oculto en Zona Verde (Método Kocher)

## Información General

**Nivel:** Intermedio-Avanzado

**Método:** Kevin Kocher (Trailing Positivo)

**Duración estimada:** 30-40 minutos por binomio

**Objetivo principal:** El perro debe (1) **marcar una puerta cerrada** por la que pasó el figurante, y (2) localizar un **figurante tumbado en el suelo cubierto con manta de camuflaje** en una zona verde con árboles y arbustos.

---

## Habilidades Entrenadas

1. **Marcaje de obstáculo**: El perro indica al guía que el rastro continúa a través de una barrera que no puede cruzar
2. **Hallazgo a nivel de suelo**: Figurante tumbado y camuflado — el perro no puede usar la vista
3. **Transición de terrenos**: Parking (asfalto) → zona verde (hierba, tierra, vegetación)

### Comportamiento de Marcaje Esperado

```
SECUENCIA DE MARCAJE EN LA PUERTA:

    🐕 Perro llega a la puerta
    │
    ├─ 1. Olfatea intensamente la puerta y marcos
    │     (detecta que el rastro continúa al otro lado)
    │
    ├─ 2. Intenta pasar (empuja, rasca)
    │     ← NORMAL, no es el marcaje aún
    │
    ├─ 3. Se frustra ligeramente
    │     (el rastro está ahí pero no puede seguir)
    │
    ├─ 4. MARCA → Señal entrenada:
    │     ┌─ Sentado mirando la puerta
    │     ├─ Rascado con pata en la puerta
    │     ├─ Toque de nariz en la puerta
    │     └─ Ladrido dirigido a la puerta
    │
    └─ 5. Guía RECONOCE el marcaje
          │
          ├─ "¡Muy bien!" (refuerzo verbal)
          └─ ABRE la puerta para el perro
             El perro CONTINÚA el rastro
```

## Procedimiento Completo

### Fase 0: Preparación del Figurante (5 minutos)

```
PREPARACIÓN:

    FIGURANTE A                        GUÍA + PERRO
    │                                       │
    ├─ Motiva al perro en el parking       ├─ Participan en motivación
    │  (juego, voz, contacto)               │
    │                                       │
    ├─ Deja artículo de olor               ├─ Se retiran FUERA
    │  (prenda personal en bolsa)           │  de vista
    │                                       │
    ├─ Camina por el parking               │
    │  hacia la puerta de la valla          │
    │                                       │
    ├─ Atraviesa la puerta                 │
    │                                       │
    ├─ ★ CIERRA LA PUERTA ★               │  ← CRÍTICO
    │  (el perro NO puede pasar)            │
    │                                       │
    ├─ Camina 50m por zona verde           │
    │  (entre árboles y arbustos)           │
    │                                       │
    ├─ Se TUMBA en el suelo                │
    │                                       │
    ├─ Se cubre con MANTA CAMUFLAJE        │
    │  (queda totalmente oculto)            │
    │                                       │
    └─ Prepara recompensa debajo           ├─ Instructor da
       de la manta (juguete + comida)       │  señal "listo"
                                            │
                                            └─ Acerca al punto
                                               de inicio
```

### Fase 1: Trabajo del Perro — Parking (3 minutos)

- Presentar artículo de olor (5-15 segundos)
- Señal verbal: "Busca"
- El perro sigue el rastro entre los coches del parking
- Avance hacia la valla

### Fase 2: Marcaje de la Puerta — Momento Clave (2-5 minutos)

- El perro llega a la puerta cerrada
- El rastro continúa al otro lado — el perro lo detecta
- **El perro MARCA la puerta** (señal entrenada)
- Guía reconoce el marcaje → abre la puerta
- El perro continúa

### Fase 3: Zona Verde — Hallazgo Final (5 minutos)

- El perro entra en la zona verde
- Cambio de terreno: asfalto → hierba/tierra
- Turbulencia de olor por vegetación — el perro trabaja más lento
- Localiza al figurante tumbado bajo la manta
- **Indica** sobre el figurante oculto

### Fase 4: Recompensa (1 minuto)

- Figurante se descubre → **celebración máxima**
- Juego + comida durante 30-60 segundos
- El perro asocia: marcaje + hallazgo oculto = éxito total

## Tabla de Comportamiento Esperado

| Punto | Comportamiento Esperado | Si No Ocurre |
|---|---|---|
| Parking | Sigue rastro entre coches | Distracciones → limpiar zona de personas/otros perros |
| Puerta cerrada | Olfatea intensamente, intenta pasar | Normal — esperar |
| Puerta cerrada | **Marca la puerta** (alerta entrenada) | No marca → entrenar marcaje de puertas como ejercicio aislado |
| Guía abre puerta | Perro retoma rastro inmediatamente | Pierde interés → figurante hace ruido suave desde el escondite |
| Zona verde | Nariz abajo, busca en vegetación | Cabeza arriba (búsqueda visual) → reducir distancia a 25m |
| Manta camuflaje | Indica sobre figurante tumbado | Pasa de largo → figurante emite sonido mínimo, reducir camuflaje |

## Progresión

| Sesión | Distancia puerta-hide | Camuflaje | Dificultad Añadida |
|---|---|---|---|
| Primera vez | 25m | Manta parcial (pies visibles) | Ninguna |
| Repetición 2 | 50m | Manta total (esta sesión) | Estándar |
| Repetición 3 | 75m | Manta + hojas/ramas encima | Vegetación densa |
| Repetición 4 | 100m | Camuflaje total + viento cruzado | Condiciones operativas |

---

## Notas Método Kocher

- **La puerta DEBE estar cerrada**: Si queda abierta, el perro pasa y no aprende a marcar. No hay aprendizaje sin obstáculo.
- **El marcaje se entrena por separado primero**: Si el perro nunca ha marcado una puerta, hacer sesiones específicas de marcaje ANTES de combinar con rastro completo.
- **Manta de olor neutro**: La manta camuflaje debe estar lavada y aireada. El único olor detectable debe ser el del figurante debajo.
- **Sin correcciones**: Si el perro no marca, NO se le obliga. Se reduce la dificultad y se reintenta.
- **Celebración ENORME al final**: Este ejercicio tiene dos dificultades (puerta + camuflaje). La recompensa debe ser proporcional al esfuerzo.
