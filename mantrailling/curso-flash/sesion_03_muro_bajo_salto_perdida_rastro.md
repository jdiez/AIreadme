# Sesión 03 — Muro Bajo, Salto y Pérdida de Rastro (Método Kocher)

## Información General

**Nivel:** Avanzado

**Método:** Kevin Kocher (Trailing Positivo)

**Duración estimada:** 35-45 minutos por binomio

**Objetivo principal:** El perro sigue un rastro largo (150m) a lo largo de un muro bajo de ladrillo, y debe **marcar el punto exacto donde el figurante saltó el muro** y desapareció al otro lado — causando la pérdida del rastro en el lado del perro.

---

## Habilidades Entrenadas

1. **Seguimiento de rastro largo** junto a una barrera (muro bajo)
2. **Marcaje de punto de salto**: El perro indica DÓNDE el figurante cruzó el obstáculo
3. **Gestión de pérdida de rastro**: El rastro desaparece abruptamente — el perro debe comunicarlo
4. **Lectura del guía**: El guía interpreta correctamente que el perro ha marcado un cambio de dirección vertical

```

    ❌ PÉRDIDA DE RASTRO (perro perdido):      ✅ MARCAJE DE SALTO (perro indica):
    │                                           │
    ├─ Movimiento errático                     ├─ Focalización en UN punto
    ├─ Nariz arriba buscando en el aire        ├─ Nariz en el MURO específicamente

                                                    ↑
                                                    │

                                               El perro dice: "AQUÍ pasó"
```

## Procedimiento Completo

### Fase 0: Colocación del Rastro por el Figurante (10 minutos)

```
SECUENCIA DEL FIGURANTE:

    FIGURANTE A                                 GUÍA + PERRO
    │                                                │
    ├─ Motiva al perro en el punto de inicio        ├─ Participan
    │                                                │
    ├─ Deja artículo de olor [👃]                   ├─ Se retiran
    │  (3m antes de la esquina)                      │  FUERA de vista
    │                                                │
    ├─ Camina hasta la ESQUINA del muro             │
    │                                                │
    ├─ Gira a la DERECHA                            │
    │                                                │
    ├─ Camina 150m pegado al muro                   │
    │  (por el lado del perro / calle)               │
    │  Paso normal, sin paradas                      │
    │                                                │
    ├─ A los 150m → SALTA EL MURO                  │
    │  (muro bajo, ~80cm)                            │
    │                                                │
    ├─ Se agacha INMEDIATAMENTE                     │
    │  al otro lado del muro                         │
    │  (1-2m de distancia)                           │
    │                                                │
    └─ Permanece oculto y quieto                    ├─ Instructor da
       Prepara recompensa                            │  señal "listo"
                                                     │
                                                     └─ Acerca al punto
                                                        de inicio
```

### Fase 1: Trabajo del Perro — Inicio y Esquina (2 minutos)

- Presentar artículo de olor (5-15 segundos)
- Señal verbal: "Busca"
- El perro detecta el rastro, llega a la esquina, gira a la derecha
- Comienza a seguir el muro

### Fase 2: Recorrido Largo — 150m Junto al Muro (5-8 minutos)

```
DURANTE EL RECORRIDO LARGO:

    🐕──────────────────────────────────────────→
    │
    ├─ El perro sigue el rastro pegado al muro
    ├─ Ritmo constante (no hay cambios de dirección)
    ├─ El guía mantiene línea SUELTA
    ├─ Posibles distracciones en la calle
    │  (peatones, otros perros, tráfico)
    ├─ El guía NO tira de la línea
    └─ Si el perro investiga una distracción → esperar
       y dejarlo volver al rastro por sí mismo
```

### Fase 3: Punto de Salto — Momento Clave (3-5 minutos)

- El perro llega al punto donde el figurante saltó
- El rastro SE CORTA abruptamente
- El perro muestra confusión → chequeo → **marcaje**
- El guía reconoce el marcaje

### Fase 4: Resolución y Recompensa (2 minutos)

- El guía lleva al perro al otro lado del muro (rodeando o ayudando)
- El perro encuentra al figurante agachado detrás del muro
- **Celebración máxima** — recompensa proporcional al esfuerzo (150m + marcaje)

## Tabla de Comportamiento Esperado

| Punto | Comportamiento Esperado | Si No Ocurre |
|---|---|---|
| Esquina | Gira a la derecha siguiendo el rastro | Duda → normal en esquinas, dar tiempo |
| Recorrido 150m | Avance constante pegado al muro | Se despega → distracciones, limpiar ruta |
| Punto de salto | Confusión, vuelve atrás, nariz en el muro | Sigue de largo → acortar a 80m, poner marca de olor en el muro |
| Punto de salto | **Marca el muro** en punto específico | No marca → reduce distancia, practica marcaje aislado |
| Otro lado del muro | Encuentra figurante agachado | Pierde motivación → figurante llama suavemente |

## Notas Método Kocher

- **El perro PIERDE el rastro**: Esto es intencional. El objetivo es que el perro **comunique** dónde desapareció el olor, no que lo resuelva solo.
- **Marcaje ≠ pérdida**: El guía debe distinguir entre un perro perdido (errático) y un perro que marca (focalizado en un punto). Esta lectura es fundamental.
- **Nunca forzar al perro más allá del punto de salto**: Si el perro marca correctamente, el ejercicio está completo. El hallazgo del figurante es la recompensa, no el objetivo del ejercicio.
- **150m es largo**: Para perros menos experimentados, empezar con 80m. La longitud del recorrido añade fatiga mental.
- **Sin correcciones**: Si el perro se pierde, se le lleva de vuelta al último punto de rastro y se le deja reintentar. Nunca se le lleva directamente al figurante.
- **El muro bajo es CLAVE**: No usar muros altos donde el perro no pueda detectar olor residual en la parte superior. El olor del salto queda en la parte alta del muro — el perro debe poder alcanzarlo con la nariz.
